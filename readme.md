## Install

> Requires Node.js 7.6+ and the Alfred Powerpack.

```shell
$ npm install -g alfred-ant-design-docs
```

Or
Downlaod workflow on this [URL](http://www.packal.org/workflow/ant-design-docs)

## Usage

```
antd divider
```

```
antd form
```

```
antd affix
```

Press `Enter` to open in web browser.
